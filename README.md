2. initial animation state -> initial={{}}
3. transition options -> transition={{}}  some transition types like spring, tween.etc

8. AnimatePresence -> make the component come out or exit smoothly
9. Animating Routes 
10. Animating SVG -> the path of svg use the PathLength
